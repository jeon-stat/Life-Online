# Life Online

Mobile-first test build for the Life Online character app.

## Current goal

- Show the user's cute 3D character on the main screen
- Let people open the shared address on mobile
- Auto-deploy to GitHub Pages whenever `main` is updated

## Files

- `index.html`: current shareable test app
- `.github/workflows/deploy-pages.yml`: Pages deploy workflow
- `test-version/index.html`: local prototype copy
